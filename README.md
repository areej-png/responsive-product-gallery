## ğŸ“Œ Project Overview

This project demonstrates how a **basic e-commerce flow** works on the frontend **without using any frameworks or backend**.  
Users can:
- Browse products
- Add items to the cart
- Manage quantities  
- Apply discount codes  
- See real-time price calculations  


The cart data is stored using **localStorage**, so items remain saved even after refreshing the page.

---

## âœ¨ Features

- Responsive navbar with hamburger menu
- Product listing with stock control
- Add to cart functionality
- Increase / decrease product quantity
- Disable quantity when stock limit is reached
- Remove items from cart
- Cart persistence using **localStorage**
- Discount coupon system: `SAVE20`, `FIRST10`, `WELCOME15`, `SUPER25` 
- Free shipping above a fixed order value
- Subtotal, discount, shipping & total calculation
- Fully responsive (mobile, tablet, desktop)
